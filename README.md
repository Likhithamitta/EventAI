# EventAI – AI-Powered Event Planning System

An AI-powered Event Planning System built using IBM BOB, IBM watsonx.ai Runtime, Python Flask, HTML, CSS, JavaScript, and SQLite. The platform helps users efficiently create, organize, and manage events with intelligent AI assistance.

---

## 🚀 Features

- AI-powered event planning assistance
- User Registration and Login
- Create and Manage Events
- Personalized Event Dashboard
- Event History Management
- Analytics and Insights
- User Profile and Settings
- Responsive and User-Friendly Interface

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone <repository-url>
cd EventAI
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure Environment Variables

Create a `.env` file using `.env.example` and add your IBM watsonx.ai credentials.

```
WATSONX_API_KEY=your_api_key
WATSONX_PROJECT_ID=your_project_id
WATSONX_URL=https://your-region.ml.cloud.ibm.com
```

### 4. Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://localhost:5000
```
