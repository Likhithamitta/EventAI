# EventAI – AI-Powered Event Planning System

## Project Overview
EventAI is an AI-powered event planning system developed using IBM BOB and IBM watsonx.ai Runtime. It helps users create, organize, and manage events with intelligent recommendations and an easy-to-use interface.

## Features
- User registration and login
- AI-assisted event planning
- Event dashboard
- Event history
- Analytics and insights
- User settings

## Technologies Used
- IBM BOB
- IBM watsonx.ai Runtime
- Python (Flask)
- HTML
- CSS
- JavaScript
- SQLite
